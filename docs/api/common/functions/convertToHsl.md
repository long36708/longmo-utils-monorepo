# Function: convertToHsl()

```ts
function convertToHsl(color): string;
```

将颜色转换为HSL格式。

HSL是一种颜色模型，包括色相(Hue)、饱和度(Saturation)和亮度(Lightness)三个部分。

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `color` | `string` | 输入的颜色。 |

## Returns

`string`

HSL格式的颜色字符串。
