# Function: $()

```ts
function $(selector): HTMLElement | null;
```

通过 CSS 选择器获取单个 DOM 元素

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `selector` | `string` | CSS 选择器字符串 (例如 '.class', '#id', 'div > p') |

## Returns

`HTMLElement` \| `null`

第一个匹配的 DOM 元素,如果未找到则返回 null

## Examples

```typescript
const header = $('header');
const nav = $('nav');
const footer = $('footer');
```

```typescript
const activeItem = $('.menu-item.active');
const submitButton = $('#submit-btn');
const firstCard = $('.card:first-child');
```

```typescript
const inputText = $('input[type="text"]');
const navLink = $('nav a.external');
const checkedCheckbox = $('input[type="checkbox"]:checked');
```

```typescript
const element = $('header');
if (element) {
  console.log(element.textContent);
}
```
