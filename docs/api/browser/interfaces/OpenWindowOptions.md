# Interface: OpenWindowOptions

打开窗口选项接口

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="noopener"></a> `noopener?` | `boolean` | 是否启用 noopener，默认为 true |
| <a id="noreferrer"></a> `noreferrer?` | `boolean` | 是否启用 noreferrer，默认为 true |
| <a id="target"></a> `target?` | `string` | 打开目标窗口，默认为 '_blank' |
