# Longmoo Utils API Documentation

Welcome to the Longmoo Utils documentation. This library provides high-quality TypeScript utility functions across different platforms.

## Packages

- **@longmo-utils/common**: Universal utility functions without platform dependencies
- **@longmo-utils/browser**: Browser-specific utility functions with DOM/BOM dependencies
- **@longmo-utils/node**: Node.js-specific utility functions

## Quick Start

```bash
# Install the package you need
pnpm add @longmo-utils/common

# Import functions
import { unique, debounce, camelCase } from '@longmo-utils/common'
```

## Features

- ✅ Fully typed with TypeScript
- ✅ Zero runtime dependencies
- ✅ Tree-shakeable
- ✅ Platform-specific optimizations
- ✅ Comprehensive documentation
